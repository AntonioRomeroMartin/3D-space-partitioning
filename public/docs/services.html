createDatasetService(deps) → {Object}

Description:

  • Factory that creates the dataset service responsible for loading, caching, colorizing, and displaying PCD point clouds in the scene.

    All datasets are cached after the first load so subsequent loads are instant. URL-based datasets (bundled or arbitrary) are keyed by their URL string; local File objects are keyed by __local__:<filename> and can be switched back to from the UI without re-reading the file.

createPartitionService(deps) → {Object}

Description:

  • Factory that creates the partition service responsible for building, caching, and querying spatial partitioning trees (Octree, k-d Tree, BSP Tree).

    Trees are cached by algorithm + datasetPath key so rebuilding is only done once per session. The service also tracks the current display depth and exposes changeDepth for keyboard-driven depth navigation.
