exports.KdTreeVisualizer

Visualizers. exports.KdTreeVisualizer

Visualizer for the k-d Tree algorithm. Colors each cell by its splitting axis: red = X, green = Y, blue = Z. The axis is determined by node.depth % 3, matching the build strategy.
