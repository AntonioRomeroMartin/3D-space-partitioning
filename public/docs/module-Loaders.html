(static) exports.loadPointCloud(scene, source, onLoaded, onError, onProgressopt)

Description:

  • Loads a PCD point cloud from a remote URL or a local File object and adds it to the scene. For local files a temporary object URL is created and revoked after loading. Parsing is performed synchronously on the main thread by Three.js's PCDLoader.
