KdTree

algorithms# KdTree

k-d Tree spatial partitioning structure.

At each level the splitting axis cycles through X → Y → Z (depth % 3). The split plane is placed at the midpoint of the current node's bounding box along the active axis, giving O(n log n) total build time. Points are partitioned into a left child (coord < splitValue) and a right child (coord ≥ splitValue); degenerate splits that produce an empty side finalize the node as a leaf instead.
